# Calculadora Web

Projeto de uma calculadora funcional desenvolvida para praticar conceitos básicos de desenvolvimento web.

## Descrição

Esta aplicação permite realizar operações matemáticas básicas através de uma interface simples e intuitiva.

## Funcionalidades

- Operações básicas: adição, subtração, multiplicação e divisão
- Interface com tema escuro
- Layout organizado em grid
- Suporte a teclado
- Tratamento de erros

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript

## Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo index.html no navegador

## Melhorias futuras

- Implementar histórico de operações
- Adicionar suporte a teclado completo
- Melhorar responsividade para dispositivos móveis
